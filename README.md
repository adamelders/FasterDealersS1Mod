# FasterDealers Mod for Schedule 1

This is a simple mod to make all drug dealer NPCs walk faster with a configurable speed. Cross-compatible with both IL2CPP (main branch) and Mono (alternate beta branch)!

## Requirements

- [MelonLoader v0.7.0+](https://thunderstore.io/c/schedule-i/p/LavaGang/MelonLoader/)
- [S1API Forked by ifBars v2.6.4+](https://thunderstore.io/c/schedule-i/p/ifBars/S1API_Forked/)

## Installation

1. Make sure you have the requirements listed above.
1. Download and install from Thunderstore Mod Manager.
   - If you want to install manually, download and extract the `FasterDealers.dll` file into your game's mod directory, e.g. `C:\Program Files\Steam\Steamapps\common\Schedule 1\Mods`

## Configuration

- You can configure this mod in the following file: `<Schedule 1 directory>/UserData/MelonPreferences.cfg`

### Default Config

```ini
[FasterDealers]
Enabled = true
SpeedMultiplier = 3.0
```
